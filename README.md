## Additional Info & Tech Stack

### Технічні навички:
* **Веб-технології:** HTML5, CSS3, JavaScript
* **Інструменти:** Git, GitHub, VS Code, Canva/Figma (UX/UI дизайн)
* **Бази даних:** SQL (базовий рівень)

### Що подобається у веб-розробці (Likes):
* Коли складна адаптивна верстка ідеально виглядає і на телефоні, і на великому моніторі.
* Проектувати логічні, інтутивно зрозумілі інтерфейси для користувачів.

### Що не подобається (Dislikes):
* Сайти з купою нав'язливої реклами та незручною навігацією, де неможливо знайти потрібну кнопку.
* Код без відступів та нормальних назв класів.

### Хобі & Захоплення:
* Вивчення сучасних трендів у UX/UI дизайні.
* Читання.
* Фітнес та здоровий спосіб життя.
* Подорожі та дослідження цікавих історичних місць.
